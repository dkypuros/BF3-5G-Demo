Closed Loop Phases (OODA Cycle):

  +------------------------------------------------------------------+
  |                    CLOSED LOOP CYCLE                             |
  +------------------------------------------------------------------+
  |  1. OBSERVE                                                      |
  |     - Query Prometheus for metrics                               |
  |     - Metrics: amf_registered_ues, smf_sessions,                 |
  |       upf_throughput, nf_response_time                           |
  |                                                                  |
  |  2. ANALYZE                                                      |
  |     - Compare observations to intent constraints                 |
  |     - Calculate deviation from targets                           |
  |     - Generate assessment (NORMAL / THRESHOLD_EXCEEDED)          |
  |                                                                  |
  |  3. DECIDE                                                       |
  |     - Check if recommendation exists                             |
  |     - Determine action type based on intent                      |
  |                                                                  |
  |  4. ACT                                                          |
  |     - Execute remediation action                                 |
  |     - SCALE_OUT: Call VNFM to scale                              |
  |     - Track action state and result                              |
  |                                                                  |
  |  [Wait observeInterval seconds, repeat]                          |
  +------------------------------------------------------------------+
