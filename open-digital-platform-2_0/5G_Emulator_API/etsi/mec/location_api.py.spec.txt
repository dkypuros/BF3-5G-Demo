INTEGRATION WITH 5G CORE
================================================================================

Location Service integrates with AMF to retrieve UE location:

  +------------------+                    +------------------+
  |   MEC App        |----Location API--->|  Location        |
  |                  |                    |  Service         |
  +------------------+                    +--------+---------+
                                                   |
                                                   | Namf-comm
                                                   v
                                          +------------------+
                                          |   AMF (amf.py)   |
                                          |   UE Context     |
                                          +------------------+

Data Flow:
  1. MEC App queries /location/v2/users/{userId}
  2. Location Service queries AMF for UE context
  3. AMF returns cell ID and estimated location
  4. Location Service returns formatted response
