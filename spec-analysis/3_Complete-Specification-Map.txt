INTRODUCTION
================================================================================

When you stand up this 5G lab end-to-end with full observability, you touch
an enormous number of networking specifications - far beyond just 3GPP. This
document maps every specification that comes into play, organized by layer
and function.

The lab creates a complete vertical slice through the networking stack:
  - Physical/Link layer concepts
  - Network layer (IP)
  - Transport layer (TCP/UDP)
  - Application protocols (HTTP, JSON)
  - Observability protocols (OpenTelemetry, Prometheus)
  - 3GPP-specific protocols (NAS, NGAP, PFCP, GTP-U)
  - Security protocols (TLS, OAuth2, JWT)
  - Time synchronization (PTP)
  - Machine learning pipelines

Each layer brings its own specifications into play.
